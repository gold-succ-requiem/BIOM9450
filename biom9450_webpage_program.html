AHICon
Australian Health Informatics Conference 2019

16 Dec, Sydney

Program

AHICon 2019 is a two-day program, over 16-17 December. Each day commences with a plenary session on key contemporary issues - all attendees are strongly encouraged to attend these seminars. Each plenary session will be followed by lunch, where attendees can mingle with speakers.

In addition to scheduled talks, attendees are encouraged to set up stalls for presenting their research. AHICon 2019 is now welcoming abstracts.
